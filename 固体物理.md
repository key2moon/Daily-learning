# 第二章    固体的结合

## 固体材料的结合形式

固体材料是由同种或异种原子结合而成,结合时各原子组成的离子或外壳中的电子相互作用而产生原子之间的作用力,这种作用力又称*结合键。*

结合键的形式决定了原子之间结合强度，进而决定了材料的宏观性能，如:机械性能、物理性能和化学性能等。

### 常见的四种结合形式：

1.离子性结合

2.共价结合

3.金属性结合

4.范德瓦尔斯结合

> 实际固体材料的结合:
> 可能兼有几种结合形式
> 可以具有两种结合之间的过渡性质

### 2-1  离子性结合

以*带电离子*作为结合单位，形成离子晶体/极性晶体

正负离子的电子分布高度局域在离子实的附近,形成稳定的满壳层结构。

最典型的离子晶体：碱金属卤化物

离子性结合是靠离子之间的*库伦吸引作用*建立的

在离子晶体的晶格中,正、负离子相间排列,是每一种离子与异号离子为邻,库伦作用的总效果是*吸引性*的

#### 离子晶体需要受力平衡。

若两个满壳层的离子相互接近到电子云发生到显著重叠时，就会产生强烈的排斥作用

可以用泡利不相容原理和托马斯费米统计方法解释

> Thomas-Fermi equation
> $\varphi -\varphi_0 = \frac{1}{2} (3 \pi^2 n)^{\frac{2}{3}}$
> $-\varphi_0$每点处电子总能量的极大值，$\varphi$正的常数，$n$电子数密度
> 电子云的动能正比于(电子云密度)2/3,相邻离子接近时发生电子云重叠使电子云密度增加,从而使动能增加,表现为强烈的排斥作用。

实际的离子晶体便是在**邻近离子间的排斥作用**增强到和**库仑吸引作用**相抵时而达到平衡。

#### 晶体系统内能的计算

晶体系统内能为所有离子间的相互吸引库仑能和重叠排斥能的总和

先考虑一个离子的*平均库仑能*（比如正离子），令$r$表示相邻离子的距离（晶格常数）
能量可以表示为$\frac{1}{2} \displaystyle \sum_{n_1 n_2 n_3}  ' \frac{q^2 (-1)^{n_1+n_2+n_3}}{4 \pi \varepsilon_0 (n_1^2 r^2+n_2^2 r^2+n_3^2 r^2)^{\frac{1}{2}}}$

 **推导过程**
 $V_0=\frac{Q_1 Q_2}{4\pi \varepsilon_0 d}$
 如以所考虑的正离子为原点,其他离子的坐标$(n_1r,n_2r,n_3r)$
 $(n_1^2 r^2,n_2^2 r^2,n_3^2 r^2)^{\frac{1}{2}}$表示其他各离子所占格电到原点的距离
 负离子格点：$n_1+n_2+n_3=$奇数
 正离子格点：$n_1+n_2+n_3=$偶数
 $(-1)^{n_1+n_2+n_3}$考虑到正负离子电荷之间大差别
 一撇 ‘ 表明$n_1,n_2,n_3$不同时为0
 离子间库伦作用为两个离子所共有，故有因子$\frac{1}{2}$

一个负离子的平均库伦能

$\frac{1}{2} \displaystyle \sum_{n_1 n_2 n_3}  ' \frac{(-q)^2 (-1)^{n_1+n_2+n_3}}{4\pi \varepsilon_0 (n_1^2r^2+n_2^2r^2+n_3^2r^2)^{\frac{1}{2}}}$

一对离子（一个原胞）的能量恰为上式的两倍
$\frac{q^2}{{4\pi \varepsilon_0 r}} \displaystyle \sum_{n_1n_2n_3}  ' \frac{(-1)^{n_1+n_2+n_3}} {(n_1^2,n_2^2,n_3^2)^{\frac{1}{2}}}=-\frac{\alpha q^2}{4\pi \varepsilon_0 r}$

和式为一无量纲的纯数值,完全决定于晶体结构。

$\alpha$（正数）称谓马德隆常数，计算马德隆常数：

$\alpha = - \displaystyle \sum_{n_1n_2n_3}  ' \frac{(-1)^{n_1+n_2+n_3}}{(n_1^2+n_2^2+n_3^2)^{\frac{1}{2}}}$

无穷多项之和，它既有正项，又有负项。若逐项相加，有可能不能得到收敛的结果。能多项相加，逐渐逼近最终值。

$- \displaystyle \lim_{N->00} \sum_{-N<n_1,n_2,n_3<N} ' \frac{(-1)^{n_1+n_2+n_3}}{\sqrt{n_1^2+n_2^2+n_3^2}}$

#### 重叠排斥能的计算

当邻近离子的电子云有显著重叠时,将出现陡峻上升的排斥作用,称为重叠排斥能，可唯象地用下列形式的势能函数表示：$be^{-\frac{r}{r_0}} or \frac{b}{r^n}$

前一种更精确，后一种在粗略分析中更简单

在 NaCl 晶格中,可以近似地只考虑最近邻间的排斥作用。正负离子情况完全相似,每个离子有6个相距$r$的离子。

一个正离子的排斥能（负离子同）：$\frac{1}{2} \times 6 \times \frac{b}{r^n}$

每个原胞（含一对正负离子的排斥能）$6 \times \frac{b}{r^n}$

#### 总内能

$U = N \left [ -\frac{\alpha q^2}{4\pi \varepsilon_0 r}+ \frac{6b}{r^n} \right ] = N \left [ -\frac{A}{r}+ \frac{B}{r^n} \right ]$

其中，$ A = \frac{\alpha q^2}{4\pi \varepsilon_0 }$ , $ B = 6b$

#### N个晶胞的总体积

$a=2r$　　　　　　$(2r)^3=8r^3$

晶胞中包含4个离子对,即四个原胞

每个原胞的体积:$2r^3$　　　　　　N个原胞的体积:$2Nr^3$

#### 求解平衡时晶体的体积和晶格常数

原子能结合成晶体的根本原因，在于原子结合起来之后整个系统具有更低的能量

*结合能*:把分散的原子(离子或分子)结合成晶体时所释放出来的能量(W)。

如果以分散的原子作为计量内能的标准,则 –W 就是结合成晶体后系统的*内能*。

#### 内能与晶体的体积有关

$U = N \left [ -\frac{\alpha q^2}{4\pi \varepsilon_0 r}+ \frac{6b}{r^n} \right ] = N \left [ -\frac{A}{r}+ \frac{B}{r^n} \right ]$            $V=2Nr^3$

设想把原子按一定的晶格结构排列起来,开始原子相距很远,然后,逐渐缩短距离,即逐渐减小体积,系统的内能逐渐下降。当体积收缩到一定程度后,排斥作用转变为主要方面,内能将转为上升,所以一般晶体内能作为体积的函数应具有下图的形式。

实际上实现体积变化,要依靠外施的力。若相应的压强设为$p$。根据功能原理,外界作功 $p(-dV)$ 等于内能变化$dU$。则 $p (-dV) = dU$

即$p=-\frac{dU}{dV}$

一般情况下,平衡时晶体受到的仅是大气压强$p_0$,由上述关系有:

$ -\frac{dU}{dV}= p_0$

这个关系确定了平衡晶体的体积$V_0$

> 由于数量级为大气压的压力对固体体积的影响很小,$p_0 \approx 0$。故平衡时 $-\frac{dU}{dV} \approx 0$ ,平衡晶格对应于图中斜率为零的点(A点),即系统内能的极小值点。因此,已知内能函数U(V)就可以通过极值条件确定平衡晶体的体积和晶格常数。

即$-\frac{dU}{dV} = 0 \Leftrightarrow -\frac{dU}{dr} = 0$

由于有$V=2Nr^3$,$ -\frac{dU}{dr} =-\frac{dU}{dV} \frac{dV}{dr}=-6Nr^2 \frac{dU}{dV}= 0$

##### 平衡时晶体的晶格常数和体积

$\frac{dU}{dr} \vert_{r=r_0}=0$                $U = N \left [ -\frac{\alpha q^2}{4\pi \varepsilon_0 r}+ \frac{6b}{r^n} \right ] = N \left [ -\frac{A}{r}+ \frac{B}{r^n} \right ]$

$-\frac{A}{r_0^2}+\frac{nB}{r^{n+1}}=0$

$r_0=(\frac{nB}{A})^{\frac{1}{n-1}}$        $V=2Nr_0^3$

##### 晶体的结合能

利用平衡条件$-\frac{A}{r_0^2}+\frac{nB}{r^{n+1}}=0$ , $B=\frac{A}{n}r_0^{n-1}$

$U = N \left [ -\frac{\alpha q^2}{4\pi \varepsilon_0 r}+ \frac{6b}{r^n} \right ] = N \left [ -\frac{A}{r}+ \frac{B}{r^n} \right ]$

$-W=U(r_0)=\left [-\frac{A}{r}+ \frac{B}{r^n} \right ]=-\frac{NA}{r_0}(1-\frac{1}{n})= -\frac{N \alpha q^2}{4\pi \varepsilon_0 r_0}(1-\frac{1}{n})$

因此,如果能确定n便可以计算结合能

##### 晶体的体变模量

弹性体变模量完全由内能函数所决定，体变模量可以写为$K=\frac{dp}{-\frac{dV}{V}}$

$dp$为压强  $-\frac{dV}{V}$为相对体积变化  代入$p=-\frac{dU}{dV}$

对于平衡晶体就得到$K=(V \frac{d^2 U}{dV^2})_{V_0}$

> 根据平衡条件可以对体变模量进行化简。
> $K=\left ( V \frac{d^2 U}{dV^2} \right )_{V_0}=V_0 \left (\frac{d^2 U}{dV^2} \right )_{V_0}=V_0 \left [ \frac{d}{dV} \ left ( \frac{dU}{dV} \right )  ]_{V_0}$
> $U = N \left [ -\frac{A}{r}+ \frac{B}{r^n} \right ]$         $V=2Nr^3$
> $K=V_0 \left [ \frac{d}{dV} \left ( \frac{dU}{dV} \right ) \right ]_{V_0}=2Nr_0^3 \left [ \frac{d}{d(2Nr_0^3)} \left ( \frac{d \left [ -\frac{NA}{r}+ \frac{NB}{r^n} \right ] }{d(2Nr_0^3)} \right ) \right ]_{r=r_0}$
> $K = \frac{r_0^3}{2} \left \{ \frac{d}{dr^3} \left [ \frac{d}{dr^3} \left ( -\frac{A}{r}+\frac{B}{r^n} \right ) \right ] \right \}_{r=r_0}$
> $\frac{d}{dr^3}=\frac{d}{dr} \frac{dr}{dr^3}=\frac{1}{3r^2} \frac{d}{dr}$
> $K = \frac{r_0^3}{2} \left \{ \frac{d}{dr^3} \left [ \frac{d}{dr^3} \left ( -\frac{A}{r}+\frac{B}{r^n} \right ) \right ] \right \}_{r=r_0}=\frac{r_0}{18} \left \{\frac{d}{dr} \left [ \frac{1}{r^2} \frac{d}{dr} \left ( -\frac{A}{r}+\frac{B}{r^n} \right ) \right ] \right \}_{r=r_0}$
> $K = \frac{r_0}{18} \left (\frac{d}{dr} \cdot \frac{1}{r^2}\right )_{r=r_0} \cdot \left [ \frac{d}{dr} \left ( -\frac{A}{r}+\frac{B}{r^n} \right ) \right ]_{r=r_0}+\frac{1}{18r_0} \left [ \frac{d^2}{dr^2} \left ( -\frac{A}{r}+\frac{B}{r^n} \right ) \right ]$
> 前一项由于平衡条件等于 0,后一项求微商
> 后一项求微商后得
> $K = \frac{1}{18r_0} \left \{  -\frac{2A}{r_0^3}+\frac{n(n+1)B}{r_0^{n+2}}  \right \}$
> 再利用平衡条件$-\frac{A}{r_0^2}+\frac{nB}{r^{n+1}}=0$ , $B=\frac{A}{n}r_0^{n-1}$ , $ A = \frac{\alpha q^2}{4\pi \varepsilon_0 }$
> $K = -\frac{(n-1)\alpha q^2}{4\pi \varepsilon_0 \times 18r_0^4}$
> $W=\frac{N \alpha \cdot q^2}{4\pi \varepsilon_0 \cdot r_0} \left ( 1-\frac{1}{n} \right )$
> 根据实验测定了晶格常数和体变模量,就可以由上式确定排斥力中的参数n。知道n,我们可以进一步求解结合能。

结论: 当两种电负性相差大的原子(如碱性金属元素与卤族元素的原子)互相靠近时,电负性小的原子失去电子,成正离子,电负性大的原子获得电子成为负离子,两种离子靠库伦引力结合在一起形成离子晶体。大部分盐类、碱类和金属氧化物属离子化合物。
